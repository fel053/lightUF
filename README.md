# UF CLP – Conversor UF a Pesos Chilenos

Sitio web estático que convierte valores de UF a CLP usando
el valor diario publicado por mindicador.cl.

## Arquitectura
- Build offline en Python
- HTML estático por cada valor UF
- Sitemap y robots generados automáticamente
- Página humana no indexable

## Build
```bash
python build.py
